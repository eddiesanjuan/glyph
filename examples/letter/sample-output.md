## Layout Standards

### Block Style (Default)
- All elements left-aligned
- Single space within paragraphs
- Double space between paragraphs
- No paragraph indentation

### Modified Block Style (Alternative)
- Date and closing right-aligned or centered
- Body paragraphs left-aligned
- Professional but slightly less formal

## Typography

- **Letterhead**: Company name 14-16pt, bold
- **Body**: 11-12pt, readable serif or sans-serif
- **Signature**: Can be script font or handwriting image
- **Notations**: 10pt, slightly lighter

## Print Specifications

- Standard letter size (8.5" x 11")
- 1" margins all around
- Letterhead typically in header area
- Second page header: recipient name, page number, date

## Digital/Email Variant

For email delivery, the letter can be formatted as:
- PDF attachment (formal)
- Inline HTML (semi-formal)
- Plain text with formatting (informal)
