# Quote Sample Output

## Visual Description

The quote renders as a comprehensive proposal document with:

### Header Section
- **Top Left**: Company logo and business information
- **Top Right**: "QUOTE" label with quote number
- **Meta Row**: Date, Valid Until, Project Name, Status badge

### Client & Job Site Section
- Two-column layout:
  - **Left**: Client information (Bill To)
  - **Right**: Job Site location (if different from client address)

### Sectioned Line Items

Each section displays as a distinct block:

#### Section Header
```
DEMOLITION & PREPARATION
Removal of existing kitchen fixtures and preparation for new installation
```

#### Section Items Table
| Description | Details | Qty | Unit | Price | Amount |


## Color Scheme (Default)
- **Accent Color**: Professional blue (#1e3a5f)
- **Section Headers**: Light accent background
- **Optional Items**: Muted styling with "OPTIONAL" badge
- **Status Badges**:
  - Sent: Blue
  - Accepted: Green
  - Declined: Red
  - Expired: Gray

## Print Optimization
- Section breaks designed for natural page boundaries
- Acceptance section always starts on fresh page
- Compact but readable at 10pt base font
- Clear visual hierarchy for scanning
