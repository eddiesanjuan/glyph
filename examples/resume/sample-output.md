## Layout Options

### One-Page (Default)
- Condensed format for early-career
- Essential sections only
- Smaller margins and font

### Two-Page (Senior/Executive)
- Full experience with details
- Projects, publications, volunteer
- More breathing room

### Creative/Portfolio
- Color accents and modern typography
- Skills as visual tags/badges
- Photo option

## Typography
- **Name**: 18-24pt, bold, can be serif or sans
- **Section Headers**: 11-12pt, uppercase, with underline
- **Body**: 10-11pt, readable serif or sans-serif
- **Contact Links**: Smaller, can use icons

## Print Optimization
- Standard letter size (8.5" x 11")
- 0.5-0.75" margins
- Black text on white for ATS compatibility
- PDF format preferred for applications
