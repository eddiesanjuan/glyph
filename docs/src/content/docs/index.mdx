## Quick Example

```html
<script src="https://sdk.glyph.you/glyph.min.js"></script>

<glyph-editor
  api-key="gk_your_key_here"
  template="quote-modern"
  data='{"client": {"name": "Acme Corp"}, "lineItems": [...]}'>
</glyph-editor>
```

That's it. Your users can now:

- See a live preview of their document
- Click on sections to select them for editing
- Type natural language commands like "Make this more professional"
- Download as PDF with one click
